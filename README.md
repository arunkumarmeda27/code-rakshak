## вҡЎ What is Code Rakshak?

**Code Rakshak** *(Hindi: аӨ°аӨ•аҘҚаӨ·аӨ• = Protector)* is an AI-powered code analysis platform that protects your codebase by:

- рҹ”Қ **Scanning** your code for security holes, bugs, and quality issues
- рҹ“Ҡ **Scoring** it on **Strength** (security) and **Fairness** (quality) вҖ” both out of 100
- рҹ“Ӣ **Generating** easy-to-understand reports in plain English (no jargon!)
- рҹ“„ **Producing** a professional downloadable PDF report
- рҹ—ӮпёҸ **Supporting** single files, multiple files, or entire project folders

Unlike traditional linters, Code Rakshak explains **what the problem is**, **why it matters**, and **exactly how to fix it** вҖ” the way a senior developer would explain it to you.

## рҹӣЎпёҸ What Code Rakshak Detects

### рҹ”ҙ Critical Issues
| Vulnerability | Example |
|--------------|---------|
| Hardcoded Passwords | `password = "admin123"` |
| Hardcoded API Keys | `api_key = "sk-abc123..."` |
| SQL Injection | `"SELECT * FROM users WHERE name='" + user + "'"` |
| DELETE without WHERE | `DELETE FROM users` |

### рҹҹ  High Issues
| Issue | Example |
|-------|---------|
| eval() / exec() usage | `eval(user_input)` |
| Shell command injection | `os.system(cmd)` |
| Insecure deserialization | `pickle.loads(data)` |
| innerHTML assignment | `el.innerHTML = userInput` |

### рҹҹЎ Medium Issues
| Issue | Description |
|-------|-------------|
| Weak cryptography | MD5, SHA1 hashes |
| Non-HTTPS URLs | `http://` instead of `https://` |
| Empty catch blocks | Silent exception swallowing |
| document.write() | XSS risk in browsers |

### рҹҹў Low / Code Quality
| Issue | Description |
|-------|-------------|
| TODO / FIXME markers | Unfinished work |
| console.log in prod | Debug leaks |
| Low comment ratio | Under-documented code |
| High complexity | Functions too long/complex |

## рҹҡҖ Quick Start

### Prerequisites

```bash
node >= 18.0.0
npm >= 8.0.0
A Gemini API key (free at https://aistudio.google.com)
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/arunkumarmeda27/code-rakshak.git
cd code-rakshak

# 2. Set up the Backend
cd backend
npm install

# 3. Configure your Gemini API key
echo "PORT=3000" > .env
echo "GEMINI_API_KEY=your_gemini_api_key_here" >> .env
echo "FRONTEND_URL=http://localhost:5173" >> .env

# 4. Start the backend
npm run dev

# 5. In a new terminal вҖ” set up the Frontend
cd ../frontend
npm install
npm run dev
```

### вң… That's it!

Open **http://localhost:5173** in your browser.

## рҹ”Ң API Reference

### `POST /api/analyze/start`
Start a new code analysis session.

```json
// Request Body
{
  "code": "def hello():\n    print('world')",
  "language": "python",       // or "auto" for auto-detection
  "filename": "main.py"
}

// Response
{
  "analysisId": "uuid-v4"
}
```

### `GET /api/analyze/:analysisId/stream`
Connect to the live analysis progress stream (Server-Sent Events).

```
data: {"phase":"parse","message":"рҹ”Қ Parsing code..."}
data: {"phase":"analyze","message":"рҹ”җ Security audit running..."}
data: {"phase":"complete","report":"...","scores":{...},"findings":[...]}
```

### `GET /api/analyze/:analysisId/report`
Download the PDF report (only available after analysis completes).

```
Content-Type: application/pdf
Content-Disposition: attachment; filename=rakshak_report_*.pdf
```

### `GET /health`
Health check endpoint.

```json
{
  "status": "ok",
  "service": "Code Rakshak API",
  "version": "2.0.0"
}
```
